# App Store メタデータ(申請時にコピペで使う)

## アプリ名(30文字まで)

```
あしあと - 行った場所を記録する旅の地図
```

- 「あしあと」単体だと類似名(足跡/足あとマップ)に埋もれるため、検索されやすい語を名前に含める
- 「行った場所」「記録」「旅」「地図」はいずれも検索ボリュームの大きい語

## サブタイトル(30文字まで)

```
ふたりで共有・制県レベルと実績で旅がたのしい
```

## キーワード(100文字まで、カンマ区切り・スペースなし)

```
旅行,記録,行ったところ,行った場所,地図,制県,経県値,都道府県,旅,夫婦,カップル,共有,思い出,ピン,足跡,塗りつぶし
```

方針:
- 「行ったところ」「旅行 記録」で検索する人が主要ターゲット(ユーザー本人の言葉)
- 「制県」「経県値」「塗りつぶし」は人気競合(制県伝説・経県値・Journeys)の指名検索の関連流入狙い
- 名前・サブタイトルに含まれる語はキーワード欄に重複させない方がよいが、
  「旅行」「記録」は重要度が高いため重複させて優先度を確保

## 説明文(ドラフト)

```
行った場所にピンを打つだけ。地図がふたりの思い出のアルバムになる旅の記録アプリです。

■ タップするだけのかんたん記録
地図をタップ、または場所を検索するだけ。場所の名前は自動で入ります。
コメントも残せるので、旅の思い出やひとことメモもいっしょに。

■ 誰と行ったかがピンの色でわかる
メンバーを登録すると、ピンの色で「誰が行った場所か」がひと目でわかります。
ひとりで行った場所、ふたりで行った場所、まだ一緒に行けていない場所。
次の旅行の計画がたのしくなります。

■ 制県レベルと実績バッジ
ピンを打つだけで、行った都道府県が自動でカウント。
47都道府県の制覇率や訪問した国の数、実績バッジで旅のモチベーションが続きます。
制県レベルはSNSでシェアできます。

■ 夫婦・カップル・友達と共有
iCloudでふたりの記録をリアルタイム共有。相手のiPhoneでも同じ地図を見て、
いっしょに書き込めます。アカウント登録は不要です。

■ かわいい塗り絵風の地図
都道府県・国ごとに色分けされた、やさしい色合いのオリジナル地図。
眺めているだけでたのしくなります。

【プレミアム(サブスクリプション)】
場所ごとに写真を無制限で残せます。写真もiCloudでふたりに共有されます。
```
